## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/Python_With_Vedant.git
```

Move into the project folder:

```bash
cd Python_With_Vedant
```

Run any Python file:

```bash
python python_1.py
```

or

```bash
python python_5.py
```
